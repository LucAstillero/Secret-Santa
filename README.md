# Secret-Santa
Ready-to-run Secret Santa application with email notifications.

This app guarantees true randomness (within the limits of RNGs). The gift distribution process follows a uniform random derangement i.e. everyone will receive a gift from someone other than themselves, at random and with the same probability for each person. This app does NOT employ a cyclic or shifted derangent algorithm (if it did, it would not be "truly" random).

To run this app it is necessary to have a gmail account with 2FA, and an App Password.
